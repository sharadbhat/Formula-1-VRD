## Getting Started
1. Install Node (v16.x) LTS (https://nodejs.org/en/download/)
2. Make sure `npm` is installed on your system
3. Clone this repository
4. Run `cd Formula-1-VRD`
5. Run `npm i`
6. Run `npm run dev`
7. Head to http://localhost:5173
